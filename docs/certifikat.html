Instalacija Certifikat Update FAQ

Kako instalirati certifikat

Certifikat je potreban kako bi Windows vjerovao aplikaciji i omogućio instalaciju.

Koraci:
1. Preuzmite certifikat: RUK64xInstaller_MSIX.cer
2. Dvoklik na certifikat.
3. Kliknite “Install Certificate”.
4. Odaberite “Local Machine”.
5. Odaberite “Place all certificates in the following store”.
6. Odaberite “Trusted People”.
7. Potvrdite instalaciju.

Nakon instalacije certifikata možete instalirati aplikaciju.
