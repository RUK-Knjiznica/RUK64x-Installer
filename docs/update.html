Instalacija Certifikat Update FAQ

Kako radi automatski update

Aplikacija RUK64x koristi Microsoftov App Installer sustav za automatske nadogradnje.

Kako funkcionira:
• Windows pamti URL .appinstaller datoteke.
• Svaki put kada pokrenete aplikaciju, Windows provjerava ima li nove verzije.
• Ako postoji nova verzija, pojavit će se obavijest o nadogradnji.
• Klikom na “Update” aplikacija se automatski ažurira.

Ako update ne stigne odmah, pričekajte 30–60 sekundi ili ponovno pokrenite aplikaciju.
